## 環境構築方法

1. リポジトリをクローン

```sh
git clone https://github.com/gizumo-education/react-study-session.git
```

2. 対象ディレクトリに移動

```sh
cd react-study-session
```

3. モジュールのインストール

```sh
npm i
```

4. サーバの立ち上げ

```sh
npm run dev
```
5. http://localhost:5173 にアクセスし、初期画面が表示されることを確認
